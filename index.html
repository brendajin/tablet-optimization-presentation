    Choosing Libraries, Shims, Polyfills, Frameworks...

  • Conflict
  • Attitude
  • Recentness
  • Readability
  • Reason
  • Ease
  • Style
  • Size
  • Conflict - How will this interact with the libraries I already use, and the functionality I already have?
  • Attitude - How satisfied am I with the authors' responses to issues and questions?
  • Recentness - How recently has this been updated? How active is the author(s)?
  • Readability - Can I understand how how it works by reading it?
  • Reason - What do I need this library for? Is it holistic or is it for one specific purpose?
  • Ease - How easy is it for me and future colleagues to use?
  • Style - Do the patterns match the rest of the style of our site?
  • Size - How big is the library?
